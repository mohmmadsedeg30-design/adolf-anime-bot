# Adolf Anime Bot 🎌

A comprehensive Telegram bot for anime information, powered by Jikan, AniList, and Trace.moe APIs.

## 🚀 Features
- **Search:** Get detailed info about any anime.
- **Top & Seasonal:** View the highest-rated and currently airing anime.
- **Random:** Discover something new with the randomizer.
- **Image Search:** Find an anime from a screenshot using Trace.moe.
- **Favorites:** Save your favorite anime to your personal list.
- **Smart Recommendations:** Get suggestions based on your favorites.
- **Quotes:** Random anime quotes for daily inspiration.
- **Multi-language Support:** Descriptions are automatically translated to Arabic.

## 🛠️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/[YOUR_USERNAME]/adolf-anime-bot.git
cd adolf-anime-bot
```

### 2. Install dependencies
```bash
pip install python-telegram-bot aiohttp
```

### 3. Set up your Bot Token
Get a token from [@BotFather](https://t.me/BotFather) and set it as an environment variable:
```bash
export BOT_TOKEN="your_telegram_bot_token"
```

### 4. Run the bot
```bash
python main.py
```
